## 🐞 Report an Issue

> ℹ️ To open the links below in a new tab, right-click (or control-click on Mac) the link and choose "Open link in new tab."

We welcome bug reports both from within and outside the organization!

- **GitHub users:** [native issue form](https://github.com/MonstaStudios/monsta_pipeline_issues/issues/new/choose)
- **Non-GitHub users:** [external bug report form](http://issue-form.monsta.my/)
- For both: Attach screenshots whenever possible and please specify which Monsta tool is affected.

_External submissions are posted by our issue bot. Please include contact info for follow-up if desired._
